## 开发流程

```
概念设计 → 原型验证 → 垂直切片 → 内容制作 → 优化打磨 → 发布
```

### 各阶段关键任务

| 阶段 | 关键任务 |
|------|----------|
| 概念设计 | GDD、核心循环、美术风格 |
| 原型验证 | 核心玩法、技术可行性 |
| 垂直切片 | 完整流程、品质标准 |
| 内容制作 | 关卡、角色、音频、剧情 |
| 优化打磨 | 性能、平衡、Bug修复 |
| 发布 | 平台适配、营销素材 |

---

## 引擎选择指南

| 场景 | 推荐引擎 | 原因 |
|------|----------|------|
| 移动游戏 | Unity | 成熟生态、跨平台 |
| 3A 大作 | Unreal | 渲染能力、工具链 |
| 独立游戏 | Godot | 开源免费、轻量 |
| VR/AR | Unity/Unreal | 平台支持完善 |
| 2D 游戏 | Godot/Unity | 2D 工具成熟 |
