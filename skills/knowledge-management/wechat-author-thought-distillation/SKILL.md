# 公众号作者思维蒸馏技能 - nuwa-skill 方法论应用

## 概述
本技能用于对微信公众号作者进行完整的思维模式蒸馏，提取其核心心智模型、决策原则、语言风格，生成可用于AI模仿的"思维图谱"。基于 nuwa-skill（女娲造人）方法论。

## 适用场景
- 研究特定投资博主的交易思维
- 模仿特定作者的写作风格
- 提取专家的决策启发式
- 构建领域专家的思维模型库

## 核心方法论

### 蒸馏八步法

#### 步骤1：语料收集 - 多源交叉验证

**场景A：用户已提供语料库（推荐）**

当用户提供Excel/CSV/Markdown等格式的语料库时，优先使用：

**1.1 语料库格式识别**
```python
# 常见格式
Excel (.xlsx)     # 微信公众号导出，含标题/内容/时间/阅读量等
CSV (.csv)        # 简化格式
Markdown (.md)    # 单篇或合并文件
JSON (.json)      # API导出格式
```

**1.2 Excel处理流程（实测 2026-04）**
```bash
# 使用 Python pandas 解析
import pandas as pd

df = pd.read_excel('微信公众号文章.xlsx')

# 常见列名映射
标准列 = {
    '标题': 'title',
    '文章内容': 'content',
    '发布时间': 'publish_time',
    '阅读': 'read_count',
    '点赞': 'like_count',
    '链接': 'url'
}

# 输出统计
print(f"总文章数: {len(df)}")
print(f"时间跨度: {df['publish_time'].min()} ~ {df['publish_time'].max()}")
print(f"总字数: {df['content'].str.len().sum():,}")
```

**1.3 语料清洗与导出**
```bash
# 每篇文章保存为独立 Markdown
for idx, row in df.iterrows():
    filename = f"{row['publish_time'].strftime('%Y%m%d')}_{row['title'][:30]}.md"
    with open(f"articles/{filename}", 'w') as f:
        f.write(f"---\ntitle: {row['title']}\n")
        f.write(f"date: {row['publish_time']}\n")
        f.write(f"url: {row['链接']}\n---\n\n")
        f.write(row['文章内容'])

# 生成合并语料库
cat articles/*.md > complete_corpus.md
```

**1.4 质量验证**
- ✅ 至少100篇文章（本案例：3207篇）
- ✅ 时间跨度 >= 1年（本案例：2017-2026，9年）
- ✅ 总字数 >= 50万字（本案例：1620万字符）
- ✅ 内容完整性：标题+正文+时间戳

**场景B：需要从网络抓取**

**问题**：微信公众号内容往往被封禁、付费、无法直接爬取

**技术限制**（2026-04 实测）：
1. **单篇文章页面**：不包含历史文章列表，无法获取其他文章链接
2. **公众号主页**：需要微信环境或扫码登录，无法通过纯HTTP访问
3. **第三方备份**：豆瓣、知乎、雪球都有滑动验证拦截
4. **搜索引擎**：百度/搜狗对微信文章收录有延迟，最新文章可能搜不到

**解决策略**：
```
1. 搜索引擎搜索："[作者名] 公众号 文章 全文"
2. 镜像站点：ETF之家、雪球转载、知乎专栏、博客园、淘股吧等
3. 语录汇总：搜索 "[作者名] 语录" "[作者名] 精华"
4. 混合方案：部分完整文章 + 大量语录 = 足够蒸馏
```

#### 步骤2：语料清洗 - 去噪提纯
**清洗规则**：
```python
移除内容：
- 广告、推广、二维码
- 登录/注册提示
- "点击阅读全文"截断标记
- 网站导航、页脚
- 其他作者的评论

保留内容：
- 作者原创正文
- 标志性开头/结尾
- 独特的表达习惯
- 数字编号的原则/规则
```

#### 步骤3：人格定位 - 身份画像
**提取维度**：

| 维度 | 提取方法 | 示例 |
|------|---------|------|
| 身份标签 | 作者自我介绍、他人评价 | A股短线交易教父 |
| 核心气质 | 高频形容词、语气词 | 江湖气、敢说真话、老大哥 |


#### 步骤4：核心心智模型 - 5大模型法则
**提取原则**：每个模型包含：
1. **核心公式**：一句话总结
2. **推演路径**：3-5步逻辑链条
3. **反常识洞见**：与大众认知不同的地方
4. **案例佐证**：原文中的具体应用

**5大模型模板**：
```markdown
#### 模型1：[领域]核心逻辑 - X决定论
**核心公式**：`结果 = A因素 + B因素 - C因素`

**推演路径**：
1. 第一步...
2. 第二步...
3. 第三步...

**反常识洞见**：
> [原文金句]

---

#### 模型2-5：以此类推
```

**常见模型类型**：
- 因果决定论（什么决定什么）
- 周期轮回论（涨跌/兴衰循环）
- 层级/鄙视链视角（社会分层）
- 系统映射论（外部系统如何映射本领域）
- 生存法则论（适者生存的规则）

---

#### 步骤5：决策启发式 - 20条原则法则
**提取标准**：
- 可执行的行动指南
- 有明确的适用场景
- 作者反复强调的规则

**分类框架**：
```markdown
### A类：基础原则（3-5条）
- 原则1：[内容]

### B类：场景应对（5-8条）
- 原则2：[内容]

### C类：风险控制（3-5条）
- 原则3：[内容]

### D类：进阶技巧（3-5条）
- 原则4：[内容]
```

**质量要求**：
- ✅ 每条都是完整的陈述句
- ✅ 有明确的"如果...那么..."逻辑
- ✅ 能直接用于决策
- ✅ 总量控制在15-25条（太少不完整，太杂记不住）


## 常见问题与解决方案

### 问题1：语料太少，不足以蒸馏
**解决**：
- 扩大搜索范围（百度、搜狗微信、谷歌）
- 降低完整度要求，重点提取语录
- 采用"核心原则 + 少量模板"的轻量蒸馏
- 标注"语料不足，蒸馏结论仅供参考"

### 问题1.5：无法获取公众号历史文章列表
**原因**：微信公众号技术限制，需要微信环境才能访问公众号主页
**解决**：
1. **请求用户提供文章列表**：让用户在微信中导出历史文章
2. **使用第三方语录整理**：博客园、淘股吧等平台常有粉丝整理
3. **按日期搜索**：如果文章标题有规律（如"YYYY-M-D 数据"），可按日期搜索
4. **接受现状**：语录整理 + 少量完整文章已足够蒸馏

### 问题2：观点前后矛盾
**解决**：
- 保留矛盾，标注"不同时间观点有变化"
- 分析观点演变的时间线和原因
- 提炼"不变的核心" vs "变化的策略"

### 问题3：模仿效果不像
**解决**：
- 增加口头禅和语气词的权重
- 提取更多"反常识洞见"（这是人格的核心）
- 增加测试用例，迭代优化
- 采用"思维链 + 风格约束"的双重Prompt
