# 爱在冰川分析风格

> 来源：微信公众号"爱在冰川"，A股游资视角的周期筹码分析风格

## 核心特征

### 1. 周期思维
- 从**筹码分布**、**资金周期**、**情绪周期**角度分析
- 不看单一事件，看事件在周期中的位置
- 关注周期阶段：上升、震荡、退潮、冰点试错

### 2. 节奏感
- 关注市场节奏：是**加速**还是**分歧**？是**高潮**还是**退潮**？
- 判断主升浪、高位震荡、补涨、退潮
- 识别高低切换信号

### 3. 性价比
- **多谈**："位置性价比"
- **少谈**："利好利空"
- 关注资金在什么位置做了什么选择

### 4. 游资视角
- 关注**龙头**、**补涨**、**高低切换**、**亏钱效应**
- 看连板晋级率、断板后的亏钱效应
- 关注资金进攻还是防守

## 语言风格

### ❌ 不说的话
- "利好刺激上涨"
- "利空导致下跌"
- "震荡整理"（太模糊）
- "建议关注"（不明确）

### ✅ 多说的话
- "资金在什么位置做了什么选择"
- "是主升加速还是高位分歧"
- "情绪在周期的哪个位置"
- "这个位置性价比如何"
- "亏钱效应在扩散还是收敛"

## 简报应用示例

### 早盘简报
```markdown
## 周期判断

**当前阶段**：上升周期第3天

**核心观点**：
- 昨日高潮后今天分歧，资金在做高低切换

**板块节奏**：
- AI 领涨开始分歧，资金流向低位的新能源
- 高位龙头开始断板，补涨标的接力

**资金态度**：
- 进攻转防守，聚焦龙头补涨方向
```

### 收盘简报
```markdown
## 周期判断

**当前阶段**：退潮期

**核心观点**：
- 亏钱效应扩散，高位股集体杀跌

**资金行为**：
- 大资金撤退，散户接盘

**情绪定位**：
- 情绪退潮中期，还未到冰点

**筹码结构**：
- 3000 点上方筹码密集，资金在换手
```

## 在定时任务中的应用

早盘/收盘简报定时任务的 prompt 中已融入此风格：

1. 数据收集后，从周期角度解读
2. 不说"利好利空"，说"资金选择"
3. 判断市场节奏（加速/分歧/高潮/退潮）
4. 给出明确的明日看点，不模棱两可
