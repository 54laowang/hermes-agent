## 概述

iTick 是一家数据代理机构，提供全球金融市场的实时行情API，涵盖股票、外汇、加密货币、指数、期货、基金等。

**官网：** https://itick.org

---

## 🔑 关键发现：两个不同的 API

### 1. 股票 API（`/stock/quote`）

**适用数据类型：**
- ✅ A股主板股票（贵州茅台、中国平安等）
- ✅ A股指数（上证指数、深证成指等）
- ✅ 港股（腾讯控股、美团等）
- ✅ 美股（苹果、特斯拉等）
- ❌ **不支持任何 ETF**

**接口地址：**
```
https://api.itick.io/stock/quote
```

### 2. 基金 API（`/fund/quote`）⭐ 新发现

**适用数据类型：**
- ✅ 美股 ETF（QQQ、SPY、IWM等）
- ❌ A股 ETF（免费套餐不支持）

**接口地址：**
```
https://api.itick.io/fund/quote
```

## API 调用方式

### 股票 API

```python
import requests

API_KEY = "your_api_key_here"

url = "https://api.itick.io/stock/quote"
params = {
    "region": "SH",  # 上证
    "code": "600519"  # 贵州茅台
}
headers = {
    "accept": "application/json",
    "token": API_KEY
}

resp = requests.get(url, params=params, headers=headers, timeout=10)
data = resp.json()

if data.get("code") == 0 and data.get("data"):
    stock = data["data"]
    price = stock.get("p")  # 当前价
```

### 基金 API（用于美股 ETF）

```python
import requests

API_KEY = "your_api_key_here"

url = "https://api.itick.io/fund/quote"
params = {
    "region": "US",  # 美股
    "code": "QQQ"  # 纳斯达克100 ETF
}
headers = {
    "accept": "application/json",
    "token": API_KEY
}

resp = requests.get(url, params=params, headers=headers, timeout=10)
data = resp.json()

if data.get("code") == 0 and data.get("data"):
    fund = data["data"]
    price = fund.get("p")  # 当前价
    iopv = fund.get("iopv")  # 基金份额参考净值
```

### 适用场景

1. ✅ **监控主板股票** — 贵州茅台、中国平安、招商银行等
2. ✅ **监控港股** — 腾讯控股、美团、小米等
3. ✅ **监控美股** — 苹果、微软、特斯拉等
4. ✅ **监控美股ETF** — QQQ、SPY、IWM等

### 不适用场景

1. ❌ **监控A股ETF** — 免费套餐不支持，需使用新浪/腾讯/东方财富

### 推荐组合

**A股ETF监控（如恒生科技ETF 513130）：**
```
新浪财经（主） → 腾讯财经（备） → 东方财富（备） → 网易财经（最后）
```
**iTick 不适用**

**主板股票监控（如贵州茅台 600519）：**
```
新浪财经（主） → 腾讯财经（备） → iTick（备） → 东方财富（最后）
```

**美股ETF监控（如 QQQ）：**
```
iTick 基金API（主） → 其他美股数据源（备）
```

## 注意事项

### 1. 选择正确的 API

| 数据类型 | 使用 API | 接口地址 |
|---------|---------|---------|
| 股票、指数 | 股票API | `/stock/quote` |
| 美股ETF | 基金API | `/fund/quote` |
| A股ETF | 不支持 | 使用新浪/腾讯/东方财富 |

### 2. 返回 null 的情况

如果 API 返回：
```json
{
  "code": 0,
  "msg": null,
  "data": null
}
```

可能原因：
1. ❌ **使用了错误的 API** — ETF 应该用 `/fund/quote` 而非 `/stock/quote`
2. ❌ **免费套餐不支持** — 如 A股ETF
3. ❌ **代码不存在** — 检查股票代码是否正确
4. ❌ **API Key 无效** — 检查 token 是否正确

### 3. 调用频率限制

**免费套餐：**
- 每分钟 5 次调用
- 建议监控频率：每分钟不超过 1 次

**网格交易监控建议：**
- 每 5 分钟调用一次（符合免费套餐限制）
- 配合其他数据源（新浪、腾讯）作为备用
