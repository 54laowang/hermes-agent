## 交叉验证流程

### 方法：使用execute_code多源验证

```python
from hermes_tools import web_search
import json

# 示例：京东（JD）收盘数据
search_queries = [
    {"query": "JD.com stock April 29 2026 close price", "source": "tradingeconomics"},
    {"query": "JD stock April 29 2026 Yahoo Finance", "source": "yahoo"},
    {"query": "JD.com close April 29 2026 MarketWatch", "source": "marketwatch"},
]

# 执行多源搜索
results = {}
for i, q in enumerate(search_queries):
    result = web_search(query=q["query"], limit=3)
    results[f"search_{i}"] = {
        "query": q["query"],
        "source": q["source"],
        "data": result.get("data", {}).get("web", [])
    }

# 输出并分析
print(json.dumps(results, indent=2, ensure_ascii=False))
```

## 美股特有验证规则

### 1. 时区验证（最关键）

```python
# 北京时间 → 美东时间转换
beijing_time = "2026-04-30 05:30"
us_eastern_time = "2026-04-29 17:30"  # 减12小时

# 收盘日期判断
if beijing_time.hour >= 4:  # 北京时间凌晨4点后
    us_close_date = beijing_time.date - 1  # 美东收盘日期 = 北京日期 - 1
```

### 2. 英文搜索优先

美股数据**必须**先用英文关键词搜索确认：

```
✅ 正确："April 29 2026 stock close S&P 500"
❌ 错误：仅用中文"美股收盘 4月29日"（可能返回A股数据）
```

### 3. 时间戳验证

**美东收盘时间**：16:00 ET = 北京时间次日04:00

```python
# 验证数据时间戳
data_timestamp = "2026-04-29 16:00 ET"  # ✅ 正确
data_timestamp = "2026-04-30 04:00 BJ"  # ✅ 正确（同一时刻）

data_timestamp = "2026-04-30 10:00 BJ"  # ❌ 错误（这是北京时间上午10点，美股未开盘）
```

## 数据冲突处理

### 冲突示例

**京东（JD）2026-04-29收盘数据**：

| 数据源 | 价格 | 时间戳 | 可信度 |
|--------|------|--------|--------|
| Trading Economics | $29.73 | 2026-04-29 16:00 ET | P0 (0.90) |
| Yahoo Finance | $29.75 | 2026-04-29 16:00 ET | P0 (0.90) |
| 新浪财经 | $30.27 | 时间不明 | P2 (0.60) |

**处理决策**：
1. Trading Economics 和 Yahoo Finance 一致（差异<0.1%）
2. 采用P0级数据源：$29.73-$29.75
3. 排除新浪财经数据（时间戳不明）

### 用户确认触发条件

```
差异 ≥ 0.1% 且多个P0级数据源不一致
→ 触发用户确认机制
→ 展示所有数据源和差异
→ 询问用户采用哪个数据源
```

## 常见陷阱

### 陷阱1：时区混淆

```
❌ 错误：
搜索："JD.com 2026年4月30日 收盘"
返回：可能返回港股数据或A股数据

✅ 正确：
搜索："JD.com April 29 2026 close"（美东日期）
验证：时间戳是否为美东16:00后
```

### 陷阱2：盘后数据误判

```
❌ 错误：
盘后价格（19:00 ET）→ 当作收盘价格

✅ 正确：
明确标注"收盘价" vs "盘后价"
盘后价格单独分析
```

### 陷阱3：ADR与港股混淆

```
京东ADR（NASDAQ: JD）vs 京东港股（HKEX: 9618）
价格差异：需核实ADR转换比例（通常1:1，但不一定）

✅ 正确做法：
明确标注交易市场
查询ADR转换比例
必要时进行价格换算
```
