# A股ST公司财务分析与借壳预期研判技能

## 简介
针对A股被实施其他风险警示（ST）公司，进行专业财务分析、亏损原因拆解、报表真实性判断以及借壳/重组概率评估的标准化分析框架。

## 分析流程

### 第一步：时间确认（强制要求）
1. 必须先获取当前系统日期，确认年报/季报披露进度
2. 根据当前时间核对财报数据的时效性，确保使用最新公开数据
3 用户明确要求："每次提供市场分析和市值数据时，必须先获取当前日期，对所有数据进行复核验证确认准确性后再输出"

### 第二步：核心数据收集
通过web_search获取以下关键信息：
- 最新年报（最新完整财年）核心财务数据
- 被实施ST的具体原因（是亏损/净资产负/审计意见/违规担保/其他）
- 连续多年业绩走势
- 最新净资产、有息负债、货币资金
- 市场上的重组/借壳传闻

### 第三步：分析框架

#### 3.1 区分ST类型
判断公司属于哪一类ST：
- **A类：经营失败型** — 连续多年亏损、净资产缩水、濒临退市（如ST亚光）
- **B类：历史问题型** — 主业已经盈利，ST只因历史违规/担保/治理等问题（如ST八菱）
- 不同类型结论完全不同

#### 3.2 亏损原因拆解（针对经营亏损型）
分解亏损结构：
```
总亏损 = 一次性减值计提（商誉/资产） + 经营性亏损 + 财务费用 + 坏账/信用减值
```
计算各部分占比，判断：
- 减值是一次性出清还是持续发生
- 经营性亏损是否为刚性（折旧/财务费用）
- 主营收入是周期性推迟还是永久性萎缩

#### 3.3 报表真实性与"洗大澡"判断
判断是否为借壳刻意做亏：

支持刻意做亏（洗大澡）的信号：
- 连续两年大额计提商誉/资产减值，一次性出清所有潜在损失
- 营收持续下滑但减值幅度远超实际需要
- 净资产刚好维持在退市线以上，一步亏透为新股东进场铺路
- 原大股东持股比例不高，有控股权变更预期

不支持刻意做亏的信号：
- 主业已经持续盈利，净资产逐年增长
- 甚至还有现金分红能力
- ST原因是历史遗留问题（如违规担保），不是经营性亏损
- 减值是偶发一次，之后不再计提

#### 3.4 借壳/重组概率评估

高概率借壳的条件：
- 市值30-50亿（理想壳规模）
- 原大股东持股比例低（<30%），控股权变更成本低
- 连续亏损导致净资产接近零/负数，原股东有强烈卖壳动机
- 资产便于剥离（双主业/传统主业可出售）

低概率借壳的情况：
- 主业持续盈利，原股东没有卖壳压力
- 市值过大或过小偏离理想壳区间
- 当前监管政策：从严审核借壳，市场化借壳近年成功率极低
- 问题已经解决即将摘帽，不需要卖壳

### 第四步：输出结构标准
1. 开头必须标注"当前日期确认：YYYY年MM月DD日"
2. 核心财务数据对比表（近年走势 + 对比同类型案例）
3. 亏损原因/ST原因拆解
4. 报表真实性判断（是否刻意做亏）
5. 借壳/重组概率评估
6. 与同类案例对比
7. 投资视角总结，明确风险提示

## 关键原则
- **不对用户做投资建议，但要清晰揭示风险等级**
- **区分"市场预期"和"实际概率"**，不否定市场传闻但要给出专业判断
- **数据要溯源**，所有核心数据来自最新年报/公告
- **退市风险必须重点提示**，当净资产接近零时明确告知

## 注意事项
1. 当前（2026年）监管环境：借壳上市等同于IPO审核，市场化第三方借壳案例极少，多数是国资内部整合
2. A股近年退市趋严，净资产为负直接退市，ST股风险必须重点提示
3. 区别"其他风险警示"（ST）和"退市风险警示"（*ST），后者风险远高于前者
