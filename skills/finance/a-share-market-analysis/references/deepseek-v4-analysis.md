# DeepSeek V4 事件分析（2026-04-24）

> 本文档记录 DeepSeek V4 发布对 AI 行业和 A 股市场的影响分析，作为后续类似事件的参考模板。

---

## 事件概要

**发布时间**：2026年4月24日

**核心突破**：
- DeepSeek V4 是全球首个在国产硬件上完成全链条训练推理的万亿级模型
- Pro 版本参数量 1.6T，Flash 版本 285B
- 上下文窗口 100 万 tokens
- API 成本约 1 元/百万 tokens（缓存命中后低至 0.02 元）

**技术意义**：
- 国产算力首次拥有"杀手级应用"作为生态锚点
- 寒武纪、华为昇腾在发布当日完成 Day 0 适配
- 证明国产芯片已具备支撑万亿级模型的能力

---

## 行业影响

### 1. 开源 vs 闭源赛道分化加速

DeepSeek V4 以开源形式发布，API 价格仅为 OpenAI 的 1/100 至 1/500。

**市场反应**：
- OpenAI 2026 年预计亏损 140 亿美元
- 港股 AI 应用公司（智谱 AI、MiniMax）股价跳水
- 市场担忧开源模型的"降维打击"会侵蚀闭源公司商业空间

### 2. 国产算力生态完成闭环验证

**适配情况**：

| 厂商 | 适配状态 | 说明 |
|------|---------|------|
| 寒武纪 | Day 0 完成 | 适配代码已开源至 GitHub |
| 华为昇腾 | Day 0 完成 | 全链条适配 |
| 海光信息 | 进行中 | ROCm 路线 |
| 摩尔线程 | 进行中 | MUSA 架构 |

**市场意义**：这是国产算力首次拥有"杀手级应用"作为生态锚点。

### 3. Token 价格进入"白菜化"通道

**价格变化**：
- DeepSeek 全系列 API 输入缓存命中价格降至原价的 1/10
- Pro 模型叠加 2.5 折限时优惠

**趋势预测**：
- 2026 年：开源倒逼，价格下降 10-50 倍
- 2027-2028 年：专用芯片替代，再降 3-5 倍
- 2028 年后：算力过剩，再降 2-3 倍

### 4. 英伟达中国市场份额萎缩

**数据**：
- 英伟达在中国 AI 芯片市场份额预计 2026 年将萎缩至 8% 左右
- 留下的真空被华为昇腾及"国产 GPU 四小龙"填补

## 风险提示

1. **技术风险**：国产芯片先进制程良率提升缓慢，成本压力大
2. **生态风险**：软件栈碎片化（CANN/MUSA/BIRENSUPA 各自为战）
3. **市场风险**：AI 概念股估值已处于高位，需警惕回调
4. **政策风险**：美国出口管制可能进一步收紧
5. **竞争风险**：开源模型快速迭代可能挤压商业化空间
