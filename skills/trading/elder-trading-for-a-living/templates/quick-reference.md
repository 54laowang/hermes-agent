# 《以交易为生》快速参考卡

## 一、三重滤网速查

```
时间框架选择：
交易时间框架 → 中间框架
放大一级 → 第一重滤网（潮汐）
缩小一级 → 第三重滤网（涟漪）

示例：
日内交易：小时图（第二重）→ 日图（第一重）→ 15分钟（第三重）
短线交易：日图（第二重）→ 周图（第一重）→ 小时图（第三重）
中线交易：周图（第二重）→ 月图（第一重）→ 日图（第三重）
```

## 二、强力指数应用

```
买入信号：
1. 第一重滤网：EMA向上（上升趋势）
2. 第二重滤网：强力指数2日EMA < 0（回调）
3. 第三重滤网：价格突破前一根K线高点

卖出信号：
1. 第一重滤网：EMA向下（下降趋势）
2. 第二重滤网：强力指数2日EMA > 0（反弹）
3. 第三重滤网：价格跌破前一根K线低点
```

## 六、情绪检查清单

**交易前问自己：**
- [ ] 我现在情绪平稳吗？（没有愤怒、恐惧、贪婪）
- [ ] 我在独立思考吗？（不受群体影响）
- [ ] 我知道为什么交易吗？（有明确计划）

**如果答案是否定的，停止交易。**

## 七、止损原则

```
硬止损：
1. 技术止损：支撑位下方 / 最近低点下方
2. 资金止损：账户资金的2%

软止损：
1. 时间止损：N天内没有按预期运行
2. 信号止损：系统给出反向信号

原则：
✓ 每笔交易必须设止损
✓ 止损只能向盈利方向移动
✓ 小止损 + 大盈利 = 盈利交易者
```
