# 以交易为生 · 艾尔德交易体系

> "成功的三大支柱：心理学、市场分析、风险管理。良好的交易记录将它们结合。"
> "华尔街没有猛虎，但金融资产正在遭遇的各种风险却猛于老虎。"

## 身份卡

我是亚历山大·艾尔德，一位精神病学家转型的交易者。
我用心理学视角理解市场，用系统化方法执行交易。
我的核心贡献是：**三重滤网交易系统、强力指数指标、2%/6%风险控制法则**。

---

## 一、心理学：交易的第一支柱

### 1.1 个体心理

**为什么交易？**
- 自由：在世界任何角落生活和工作
- 独立：从日常琐事中解放出来
- 挑战：用思考赚钱的智力游戏

**现实与幻觉：**
- 业余者盯着报价屏幕，仿佛上面有数百万美元触手可及
- 当他伸手去交易——结果却赔了钱
- 他再交易——结果却赔了更多

**自我毁灭：**
> "生活中的大多数失败是缘于自毁。我们在自己的专业、个人生活和事业上的失败并不是由于我们的运气太差，也不是因为我们没有能力，而是因为我们潜意识中的自毁倾向。"

**匿名戒酒会的启示：**
- 承认自己无力控制市场（就像酒鬼无力控制酒精）
- 依靠系统和纪律，而不是意志力
- 一天一天地执行交易计划

### 1.2 群体心理

**市场是什么？**
- 看涨者与看跌者共同组成松散的市场
- 每个价格代表市场某个时点的群体共识
- 交易者实际上都是在赌群体的未来观点和情绪

**为何加入群体？**
- 原始冲动：为了安全希望加入群体
- 未来不确定性越大，越希望追随领袖
- 华尔街没有猛虎，但风险猛于老虎

**群体心智：**
> "当人们聚集而成为一个群体的时候，无论组成群体的个体是什么样的，他们的生活方式是否相同，他们的职业、性格或才智有何差异，已加入群体的事实会使他们形成群体思维，他们感觉、思考和行为的方式都将会与他们独处时大相径庭。"

**冰川视角解读：**
- 加入群体后，独立思考能力显著降低
- 群体情绪在恐惧与欢乐、惊慌与兴奋之间转变
- 成功的交易者必须独立思考
- **情绪周期规则**：上升周期人多去哪里，下降周期人多扭头跑

## 二、技术分析：交易的第二支柱

### 2.1 大道至简

**五种数据足矣：**
- 开盘价、最高价、最低价、收盘价、成交量
- 在这五种数据之上再堆加大量指标和图形只会徒增困扰
- **冰川共鸣**：股市最大的规律是没有规律，大道至简

### 2.2 强力指数指标（Force Index）

**核心思想：**
将价格和成交量结合，发掘每一次上涨或下降过程中多头或空头的力量。

**三条信息汇集：**
1. 价格改变的方向
2. 价格改变的程度
3. 价格改变过程中的成交量

**计算公式：**
```
强力指数 = 今日成交量 × (今日收盘价 - 昨日收盘价)
```

**应用方式：**
- **2日EMA**：寻找进入时机
  - 上升趋势中，2日EMA变负时买入（回调买入）
  - 下降趋势中，2日EMA变正时卖出（反弹卖出）
  
- **13日EMA**：确认趋势
  - 穿越到中心线上方：多头占据优势
  - 穿越到中心线下方：空头占据上风
  - 与价格背离：重要转折点

**冰川视角解读：**
> "缩量下杀，资金接不到货。站在资金角度想：想要筹码，低吸吸不到，怎么办？只能向上扫。强力指数指标就是帮你识别资金扫货的时机。"

### 2.3 移动平均

**EMA（指数移动平均）：**
- 更重视近期价格，反应更快
- 22日EMA：月度趋势
- 50日EMA：季度趋势
- 200日EMA：年度趋势

**交易准则：**
- EMA向上倾斜：上升趋势，做多
- EMA向下倾斜：下降趋势，做空或观望
- EMA走平：震荡市场，等待突破

## 三、交易系统：三重滤网

### 3.1 核心思想

**问题：**
- 趋势跟随指标（MA、MACD）在震荡市频繁假信号
- 震荡指标（RSI、随机指标）在趋势市过早反转

**解决方案：**
三重滤网将不同时间框架和不同类型指标结合，每层滤网过滤掉一层假信号。

### 3.2 第一重滤网：市场潮汐

**时间框架：**
选择你最喜欢的时间框架作为中间框架，然后放大到更高一级时间框架作为第一重滤网。

**工具：**
趋势跟随指标（EMA、MACD）

**判断：**
- EMA向上倾斜：上升趋势，只允许买入
- EMA向下倾斜：下降趋势，只允许卖出
- EMA走平：震荡市场，观望

### 3.3 第二重滤网：市场波浪

**时间框架：**
回到中间时间框架

**工具：**
震荡指标（强力指数、随机指标、RSI）

**判断：**
在第一重滤网确认的趋势方向上，等待震荡指标给出回调信号：
- 上升趋势中，震荡指标超卖：买入信号
- 下降趋势中，震荡指标超买：卖出信号

### 3.4 第三重滤网：市场涟漪

**时间框架：**
缩小到更低一级时间框架

**工具：**
价格突破

**执行：**
- 买入：在最近一根K线的高点上方设置买单
- 卖险：在最近一根K线的低点下方设置止损

**冰川视角解读：**
> "三重滤网就是周期思维的实战应用。第一重看大周期（潮汐），第二重看中周期（波浪），第三重看小周期（涟漪）。大周期定方向，中周期找位置，小周期执行。"

## 四、风险管理：交易的第三支柱

### 4.1 铁三角

**三条原则：**
1. **保护性止损**：每笔交易都必须设置止损
2. **2%法则**：单次交易最大亏损不超过账户的2%
3. **6%法则**：当月最大回撤不超过6%

### 4.2 2%法则：躲避鲨鱼袭击

**鲨鱼的一击：**
单次交易出现灾难性亏损，就像被鲨鱼咬去一条胳膊或腿。
新手一次交易亏四分之一资金，需要33%收益才能回本。

**2%原则：**
> "单次交易最大风险控制在账户资金的2%以内。"

**计算示例：**
- 账户资金：5万美元
- 最大风险：1000美元（2%）
- 买入价格：50美元
- 止损价格：48美元（每股风险2美元）
- 最大仓位：1000 ÷ 2 = 500股

**冰川共鸣：**
> "活下来比赚多少更重要。市场永远有机会，本金亏没了就结束了。"

### 4.3 6%法则：躲避食人鱼攻击

**食人鱼的攻击：**
食人鱼单独不危险，但成群攻击时致命。
连续小亏损累积起来，同样致命。

**6%原则：**
> "当月最大回撤控制在账户资金的6%以内。"

**执行方式：**
- 当月亏损达到6%：停止交易，观望
- 下个月重新开始
- 亏损时不要加大赌注，退后一步思考

**冰川视角：**
> "不确定时空仓。亏钱时更好的选择是退后一步，停下来去思考。"

### 4.4 止损设置

**硬止损：**
- 技术止损：支撑位下方、最近低点下方
- 资金止损：账户资金的2%

**软止损：**
- 时间止损：如果价格在N天内没有按预期运行，退出
- 信号止损：当交易系统给出反向信号，退出

**止损原则：**
- 不要随意移动止损（只能向盈利方向移动）
- 止损是成本，不是失败
- 小止损 + 大盈利 = 盈利交易者

## 五、交易记录：第四支柱

### 5.1 为什么需要记录？

> "保持良好的交易记录习惯，能帮助你从自己的经验中成长进步。它帮助你打破总是小赚大亏的恶性循环（就像是在滚筒中奔跑的松鼠）。"

### 5.2 每日功课

**盘前：**
- 检查持仓，调整止损
- 扫描潜在交易机会
- 回顾交易计划

**盘中：**
- 记录交易执行情况
- 观察市场行为
- 控制情绪

**盘后：**
- 更新交易日志
- 分析当日交易
- 总结经验教训

### 5.3 交易日志

**必记内容：**
1. 交易日期和时间
2. 交易品种和方向
3. 进场价格和理由
4. 止损价格和止盈目标
5. 出场价格和理由
6. 盈亏金额和百分比
7. 交易评分（A/B/C/D）
8. 情绪记录
9. 经验教训

**交易评分：**
- **A级交易**：完全按计划执行，盈利
- **B级交易**：基本按计划执行，小亏小赚
- **C级交易**：偏离计划，侥幸盈利
- **D级交易**：严重偏离计划，亏损

**冰川共鸣：**
> "站在资金角度思考：如果我是大资金，现在想要筹码，会怎么做？记录每一笔交易，就是记录资金的行为。"

## 六、冰川视角整合

### 6.1 周期思维 + 三重滤网

**艾尔德的三重滤网** = **冰川的周期思维**

- 第一重滤网（潮汐）= 大周期定位
- 第二重滤网（波浪）= 中周期找位置
- 第三重滤网（涟漪）= 小周期执行

**应用：**
> "春季行情涨得越多，4月底杀得越惨。用三重滤网：第一重看年度趋势（EMA200），第二重看月度趋势（EMA50），第三重等回调买入。"

### 6.2 筹码思维 + 强力指数

**艾尔德的强力指数** = **冰川的筹码思维**

- 强力指数为正 + 放量 = 多头强势扫货
- 强力指数为负 + 放量 = 空头强势派发
- 强力指数背离 = 资金分歧，趋势转折

**应用：**
> "缩量下杀，强力指数为负但成交量萎缩，说明资金接不到货。站在资金角度想：想要筹码，低吸吸不到，只能向上扫。"

### 6.3 情绪周期 + 群体心理

**艾尔德的群体心理** = **冰川的情绪周期**

- 群体乐观高潮 = 情绪周期高潮后 → 持股不开仓
- 群体悲观冰点 = 情绪周期冰点后 → 做两头（高标+错杀）
- 群体分化 = 情绪周期分化 → 去弱留强

**应用：**
> "当所有人都看多时，市场除下跌外别无选择。当所有人都看空时，市场除上涨外别无选择。"

### 6.4 风险优先 + 2%/6%法则

**艾尔德的2%/6%法则** = **冰川的风险优先**

- 2%法则 = 单次交易不死（躲避鲨鱼）
- 6%法则 = 当月回撤可控（躲避食人鱼）
- 不确定时空仓 = 达到6%红线停止交易

**应用：**
> "活下来第一。4-19魔咒期间，如果当月回撤接近6%，停止交易，等节后再战。"

## 七、实战清单

### 7.1 交易前检查

**心理学：**
- [ ] 情绪平稳，没有愤怒、恐惧、贪婪
- [ ] 独立思考，不受群体影响
- [ ] 知道自己为什么交易

**市场分析：**
- [ ] 第一重滤网：大周期趋势方向确认
- [ ] 第二重滤网：中周期震荡指标到位
- [ ] 第三重滤网：小周期突破信号准备

**风险管理：**
- [ ] 止损价格确定
- [ ] 仓位计算符合2%法则
- [ ] 当月回撤未达6%

### 7.2 交易中执行

**进场：**
- [ ] 按计划价格进场
- [ ] 立即设置止损
- [ ] 记录进场时间和理由

**持仓：**
- [ ] 不随意移动止损（只能向盈利方向）
- [ ] 观察强力指数变化
- [ ] 控制情绪，不看盘过度

**出场：**
- [ ] 止损触发：无条件退出
- [ ] 止盈触发：按计划退出
- [ ] 信号反转：退出观望

### 7.3 交易后总结

**记录：**
- [ ] 更新交易日志
- [ ] 评分（A/B/C/D）
- [ ] 记录情绪变化
- [ ] 总结经验教训

**反思：**
- [ ] 是否按计划执行？
- [ ] 哪些做得好？
- [ ] 哪些需要改进？
- [ ] 下次如何做得更好？

## 八、核心公式速查

### 8.1 强力指数

```
强力指数 = 成交量 × (今日收盘 - 昨日收盘)

2日EMA：短期买卖点
13日EMA：趋势确认
```

### 8.2 仓位计算

```
最大风险 = 账户资金 × 2%
每股风险 = |买入价 - 止损价|
最大仓位 = 最大风险 ÷ 每股风险

示例：
账户：5万，买入价：50，止损：48
最大风险 = 50000 × 2% = 1000元
每股风险 = 50 - 48 = 2元
最大仓位 = 1000 ÷ 2 = 500股
```

### 8.3 当月回撤控制

```
当月亏损 ÷ 账户资金 < 6%

达到6% → 停止交易，观望
下月重新开始
```

## 九、典型错误与纠正

### 9.1 心理错误

| 错误 | 表现 | 纠正 |
|------|------|------|
| 追涨杀跌 | 看到上涨就追，看到下跌就割 | 等回调，用三重滤网 |
| 报亏损 | 亏损加仓，试图摊平成本 | 止损认赔，绝不加仓 |
| 过度交易 | 频繁进出，止损累积 | 达到6%停止，观望 |
| 不设止损 | 亏损不止损，变成长期持有 | 每笔交易必须设止损 |

### 9.2 技术错误

| 错误 | 表现 | 纠正 |
|------|------|------|
| 指标堆砌 | 使用太多指标，信号冲突 | 大道至简，三重滤网 |
| 时间框架混乱 | 在不同时间框架间跳跃 | 固定时间框架体系 |
| 忽视成交量 | 只看价格，不看成交量 | 用强力指数结合量价 |

### 9.3 风险管理错误

| 错误 | 表现 | 纠正 |
|------|------|------|
| 重仓交易 | 单笔仓位过大 | 2%法则限制 |
| 不止损 | 亏损不止损 | 强制止损 |
| 亏损加码 | 亏损后加大仓位 | 6%法则停止 |
