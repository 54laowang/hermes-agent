# Stock Watcher - 自选股监控助手

## 核心功能

自选股管理与实时监控，支持添加、删除、列表、行情摘要。

## 使用方法

### 添加自选股

用户说："添加自选股 600053 九鼎投资"

**前置检查（必须执行）**：

1. **代码格式验证**：
   - 沪市主板：600xxx、601xxx、603xxx
   - 深市主板：000xxx、001xxx
   - 深市中小板：002xxx
   - 科创板：688xxx
   - 创业板：300xxx、301xxx
   - 北交所：8xxxxx、4xxxxx

2. **代码有效性检查**：
   ```
   调用 get_market_data 获取该股票最近交易日数据
   - 成功返回数据 → 股票代码有效
   - 返回空数据/错误 → 提示用户"股票代码无效或已退市"
   ```

3. **重复检查**：
   ```
   读取 watchlist.txt，检查是否已存在相同代码
   - 已存在 → 提示用户"该股票已在自选股列表中"
   - 不存在 → 继续添加
   ```

4. **名称匹配验证**（可选）：
   ```
   如果用户提供了股票名称，与实际名称对比
   - 不匹配 → 警告用户"代码与名称不符，实际名称为XXX"
   ```

**执行**：将股票代码和名称写入 `~/.hermes/stock_watcher/watchlist.txt`

---

### 查看自选股

用户说："查看我的自选股"

**执行流程**：

1. 检查 `watchlist.txt` 是否存在
   - 不存在 → 提示"您还没有添加自选股"

2. 读取列表，检查是否为空
   - 为空 → 提示"自选股列表为空，请先添加"

3. 逐只获取实时行情（含错误处理）

---

### 删除自选股

用户说："删除自选股 600053"

**执行流程**：

1. 检查代码是否在列表中
   - 不存在 → 提示"该股票不在您的自选股列表中"

2. 从 `watchlist.txt` 中移除

---

### 行情摘要

用户说："自选股行情摘要"

**执行流程**：

1. 读取自选股列表
2. 判断当前市场状态
3. 调用行情接口（含降级处理）
4. 输出摘要

---

## 市场状态判断

**时间锚定原则**：所有行情查询必须先判断市场状态

### A股交易时段

| 时段 | 北京时间 | 状态 |
|------|---------|------|
| 盘前 | 09:15-09:25 | 集合竞价 |
| 早盘 | 09:30-11:30 | 连续竞价 |
| 午间 | 11:30-13:00 | 休市 |
| 午后 | 13:00-15:00 | 连续竞价 |
| 收盘 | 15:00后 | 已收盘 |

### 美股交易时段（需换算美东时间）

| 时段 | 美东时间 | 北京时间 |
|------|---------|---------|
| 盘前 | 04:00-09:30 | 16:00-21:30 |
| 正常交易 | 09:30-16:00 | 21:30-次日04:00 |
| 盘后 | 16:00-20:00 | 次日04:00-08:00 |

**注意**：美股收盘日期 = 北京日期 - 1（凌晨04:00后）

### 非交易时段处理

```
当查询时间为非交易时段：
1. 提示用户当前市场状态（已收盘/未开盘）
2. 返回最近一个交易日的收盘数据
3. 标注数据时间戳："截至 YYYY-MM-DD 收盘"
```

## 行情获取降级策略

**必须按优先级尝试**：

```
Level 1: get_market_data (vibe-trading MCP)
   ↓ 失败
Level 2: 同花顺网页抓取
   https://stockpage.10jqka.com.cn/{stock_code}/
   ↓ 失败
Level 3: 新浪财经API（备用）
   https://hq.sinajs.cn/list={stock_code}
   ↓ 失败
Level 4: 返回缓存数据（如果有）
   ↓ 无缓存
Level 5: 返回错误提示
   "无法获取 {股票名称}({股票代码}) 的行情数据，请稍后重试"
```

**错误信息必须包含**：
- 股票代码和名称
- 失败原因（如"网络超时"、"接口限流"）
- 建议操作（如"请稍后重试"）

- 格式: `股票代码|股票名称`（每行一只）
- 最大数量: 建议50只以内（性能考虑）

## 边界条件清单

### 添加自选股前必须检查

- [ ] 股票代码格式正确
- [ ] 股票代码有效（有交易数据）
- [ ] 不在自选股列表中（防重复）
- [ ] 名称与代码匹配（如用户提供了名称）

### 行情获取时必须处理

- [ ] 市场是否在交易时段
- [ ] 主接口失败时降级到备用接口
- [ ] 所有接口失败时返回友好错误
- [ ] 非交易时段返回最近收盘数据

### 输出时必须标注

- [ ] 数据时间戳（日期 + 时间节点）
- [ ] 数据来源（同花顺/新浪财经）
- [ ] 市场状态（交易中/已收盘/未开盘）

---

## 注意事项

1. **主要支持A股市场**（沪市/深市/科创板/创业板）
2. **实时行情从多源获取**，按优先级降级
3. **数据仅供参考**，不构成投资建议
4. **严格时间锚定**，避免用过时数据
5. **所有错误友好提示**，不暴露技术细节给用户
