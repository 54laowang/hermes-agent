```
学习/决策                     信息保存
    ↑                              ↑

    ↓                              ↓
【无限试错框架】            【时空锚点原则/方法】
    ↓                              ↓
         ←——————————→
              ↓
      【时间线收敛推理】
              ↓
```

### 技能之间的关系

1. **学习环路**：试错原刑 → 无限试错框架 → 时间线收敛推理

2. **信息保存环路**：记忆保存原则 → 时空锚点原则/方法

3. **交叉连接**：
   - 无限试错框架 依赖 记忆保存原则 (记录试验结果)
   - 时间线收敛推理 依赖 试错原则 (验证假设)

## 使用指南

### 快速入门

1. 如果你需要在安全环境中进行实验，使用 **无限试错框架**

2. 如果你需要保存重要信息以便未来使用，使用 **时空锚点法**

3. 如果你需要分析复杂系统中的因果关系，使用 **时间线收敛推理**

### 常见工作流

**优化和反馈循环**
```
定义问题 → 无限试错框架 → 记录结果 → 分析模式 → 优化策略 → 重复
```

**长期项目管理**
```
知识积累 → 记忆保存原则 → 时空锚点方法 → 定期验证 → 更新备份
```

**复杂系统分析**
```
识别组件 → 时间线收敛推理 → 找出敏感节点 → 预测收敛结果 → 制定干预策略
```
