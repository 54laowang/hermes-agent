## 一、标题命名规律

### 1.1 主要标题格式

| 格式类型 | 数量 | 占比 | 示例 |
|---------|------|------|------|
| 日期+主题 | 2848篇 | 88.8% | `2018-5-16 短线资金流向`、`2018-7-22 个股公告汇总` |
| 三个包子 | 230篇 | 7.2% | `三个包子`、`2018-12-27 三个包子` |
| 纯主题 | 126篇 | 3.9% | `铁粉的福利`、`断头铡刀`、`不要慌外围大跌这么办` |
| 两个包子 | 2篇 | 0.1% | `两个包子` |

### 1.2 标题特征

- **日期格式灵活**：`YYYY-M-D` 或 `YYYY-MM-DD`，月份和日期不补零
- **包子系列**："三个包子"是核心资讯栏目，"两个包子"较少出现，偶有"三缺一包子"
- **主题直白**：标题直接点明内容，如`断头铡刀`、`给各位发个红包`
- **口语化倾向**：部分标题使用口语表达，如`不要慌外围大跌这么办`、`你举报我干啥？`

## 四、段落结构与节奏

### 4.1 数据统计

- **平均段落长度**：262.67字符
- **短段落数量**（<50字符）：3431个
- **长段落数量**（>500字符）：247个

### 4.2 分隔符使用

主要使用连续斜线作为分隔符：
```
////////////////////////////////////////////////////////
```

以及短分隔符：
```
-------------------------------------------------
```

### 4.3 结构特征

1. **模块化**：文章常用分隔符切分为独立板块
2. **数据密集**：公告类文章段落较短，数据列表为主
3. **个人文章**：段落较长，叙事性强
4. **节奏变化**：信息性段落与吐槽段落交替

## 五、幽默方式与比喻习惯

### 5.1 自嘲式幽默

```
一顿操作猛如虎，还不够昨天给大爷敬茶赔的钱
```

```

```

```

```

```

```

### 5.2 比喻表达

| 原意 | 比喻表达 |
|-----|---------|
| 股价最高点卖出 | 卖飞 |
| 买入后下跌被套 | 套牢、吃面 |
| 下跌时买入 | 抄底、接飞刀 |
| 集中卖出导致下跌 | 杀跌 |
| 连续上涨后大跌 | 断头铡刀 |
| 买入后立即下跌 | 中计了 |
| 看好但不敢买 | 知行不合一 |
| 持续亏损麻木 | 亏麻了 |

### 5.3 网络用语

- `卧槽` - 惊讶感叹
- `牛逼` - 赞叹
- `特码的` - 脏话替代
- `咔咔咔` - 形容走势
- `猛撸` - 形容大幅拉升
- `show` - 炫耀/尴尬

## 六、禁忌词与回避表达

### 6.1 含蓄表达

| 直白表达 | 回避/替代表达 |
|---------|-------------|
| 推荐买股 | 思考过程、供参考 |

```

```

```

```

```

- `大概率`、`有机会`
- `如果...那么...`

## 九、个人风格标签

### 9.1 核心特征

1. **数据导向**：大量使用数字、百分比、技术指标
2. **口语化表达**：夹杂网络用语、自嘲、感叹
3. **互动性强**：频繁使用问句、称呼读者
4. **情绪真实**：不掩饰亏损、分享真实感受
5. **免责意识强**：反复声明非荐股

### 9.2 语言指纹

- **称呼词**：`川哥`（被粉丝称呼）、`老铁`、`客官`、`诸君`、`各位`
- **语气词**：`哈`、`哟`、`呢`、`啦`、`嘛`
- **转折词**：`但是呢`、`当然`、`只是`、`那么`
- **强调词**：`非常`、`特别`、`超级`、`真的`

### 9.3 独特表达

```
我有种预感
```

```
华尔街没有新鲜事
```

```
很明确的告诉你
```

```

## 十、写作模式总结

### 10.1 公告资讯类（占比约90%）

- 标题：日期+主题
- 内容：数据列表为主
- 特点：信息密集、结构固定
- 语气：客观、官方

### 10.2 个人观点类（占比约10%）

- 标题：纯主题或吸引眼球
- 内容：分析+感悟+互动
- 特点：口语化、情绪化
- 语气：亲切、幽默、真实

### 10.3 时间规律

- **每周五**：惯例"瞎扯淡"
- **节假日前**：特别更新
- **重大事件**：急速策略更新

---

## 十一、风格模仿要点

1. **开场**：用"周五惯例"或直接进入主题
2. **数据**：穿插具体数字和百分比
3. **转折**：善用"但是呢""当然""那么"
4. **自嘲**：适时加入"又开始爆亏了"类表达
5. **互动**：用问句引导思考
6. **收尾**：加免责声明
